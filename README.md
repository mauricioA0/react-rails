### Installation

Requires [Node.js]

Requires [Ruby] and [Rails]

Install the dependencies and devDependencies and start the server.

```sh
$ cd server
$ bundle install
$ rails server
```

Install the dependencies and devDependencies and start the client.

```sh
$ cd client
$ npm install
$ npm run start
```
